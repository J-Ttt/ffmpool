Miner Configuration

This is the basic guide how to setup your miner to this pool.

Getting started

To get started mining and use this pool you need the following

  • {{ coinName }} Wallet address
  • Mining software that supports the {{ coinName }} coin and algorithm {{ aglorithm }}
  • Hardware to run it on. This can be your home PC, mining rig or cloud mining for example.

{{ coinName }} Wallet address

A wallet address is needed to payout you shares mined at this pool server. Get your wallet for Bamboo here
When the total mined value is past the payout threshold, we will send your coin to this wallet address.


Mining Software

To mine at this pool you can use any miner supporting the {{ aglorithm }} algorithm or {{ coinName }} coin.
Currently only one miner is supported download here. Download the miner from releases and configure the miner.

Open config.json with your favorite text editor

  • POOL STRATUM ADDRESS AND PORT - one off the stratum addresses above in the Pool Configuration
  • YOUR_WALLET_ADDRESS - your valid {{ coinName }} wallet address
  • WORKERNAME - an optional workername can be used to identify the miner or RIG
  • PASSWORD - use x or leave it blank


Example Configuration in config.json
    "_url": "Pool or node url. You can set multiple addresses, first one will be used as default and if connection fails miner will move to next one. Address prefix determines protocol used.",
    "url": [
        "stratum+tcp://bmb.ffmpool.com:4444"
    ],
    "_user": "Username or wallet address as required by pool",
    "user": "007B7857DBA9E4AFD1E53AFAF2155D6EB088B2D4AA4CEF3AF8.MyFirstWorker",
    "_password": "Password",
    "password": "MyPassword",
